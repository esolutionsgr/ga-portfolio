WELCOME TO SYDNEY WDI

On Monday 17th of June 2013, General Assembly started their first Web Immersive Development 12 WEEK COURSE IN SYDNEY

WHAT WE LEARNT

RUBY ON RAILS

We learned how to build custom apps by integrating routing, controllers, views, and databases, integrate forms into web application and authentication (login).

GITHUB

We manage group work on Github and can even serve apps via Herouku.

AJAX & jQuery

WDI focuses on learning to write JavaScript (JS) that allows the browser to communicate with the server without reloading the current page, to do things like validate or save form input and refresh images.

TEST DRIVEN DEVELOPMENT

We can now build functionality based on tests by applying Test Driven Development techniques (TDD/BDD) using RSpec. Write unit tests, functional tests and integration tests.

JAVASCRIPT

By using client-side scripts like Underscore and Backbone to interact with the user, we can control the browser, communicate asynchronously, and alter the document content that was displayed.

COMPUTER SCIENCE

We can create more efficient and elegant solutions to problems by applying fundamental Computer Science concepts to applications to differentiate between various data structures and their purpose (linked lists, stacks, hashtables, etc.).

BACKEND & API

We are trained with MySQL and PSQL databases. We can apply API's and create geolocation services apps which use open API's.

DATABASE & MVC

The class taught us how to describe and experiment with various relational Database Solutions (Postgres, MySQL, SQL) as well as describe and apply the MVC framework and how models work, experimenting with creating models for a web app.
