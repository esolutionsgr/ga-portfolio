WELCOME TO SYDNEY WDI

In 2013 #General Assembly started their first Web Immersive Development 12 WEEK COURSE IN SYDNEY

WHAT WE LEARNT

RUBY ON RAILS

“Ruby on Rails is a breakthrough in lowering the barriers of entry to programming. Powerful web applications that formerly might have taken weeks or months to develop can be produced in a matter of days.” -Tim O'Reilly, Founder of O'Reilly Media.

GIT HUB

We are trained with MySQL and PSQL databases. We can apply API's and create geolocation services apps which use open API's. We manage group work on Github and can even serve apps via Herouku.

RESPONSIVE WEB

We apply the latest HTML5 and CSS3 to Responsive web design, an approach aimed at crafting sites to provide an optimal viewing experience—easy reading and navigation with a minimum of resizing, panning, and scrolling—across a wide range of devices and we can use Twitter Bootstrap and Foundation.

JAVASCRIPT

JavaScript (JS) We use client-side scripts like Underscore and Backbone to interact with the user, control the browser, communicate asynchronously, and alter the document content that was displayed. More recently, JS has become common in server-side programming, game development and the creation of desktop applications.
